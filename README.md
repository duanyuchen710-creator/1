# 泡泡爪宠物洗护店

这是一间给猫猫狗狗准备的线上洗护门店。

项目用一个温柔、干净、可信赖的官网，把宠物洗澡、美容修剪、皮毛护理和到店预约串在一起。访客打开页面后，可以先看到门店主张，再浏览不同洗护套餐、了解服务流程、查看门店环境，最后提交预约信息。

它不像一个冷冰冰的价格表，更像店门口那块会发光的小招牌：告诉主人这里会认真看皮肤状态、照顾宠物情绪，也会把洗护过程安排得清清楚楚。

## 这个项目在做什么

- 展示宠物洗护门店的品牌首页
- 介绍基础洗护、全身精修、皮毛舒缓护理等服务
- 呈现健康询问、毛发评估、分区洗护、交接反馈等到店流程
- 展示接待区、洗护区、修剪区等门店环境
- 提供预约表单，让用户留下称呼、电话、宠物类型、服务项目和备注

## 技术栈

- Next.js
- React
- 原生 CSS

## 本地运行

```bash
npm install
npm run dev
```

然后在浏览器打开：

```text
http://localhost:3000
```

## 项目结构

```text
app/
  globals.css      页面样式
  layout.jsx       页面基础布局
  page.jsx         首页内容
components/
  BookingForm.jsx  预约表单组件
public/assets/     门店环境图片资源
```
